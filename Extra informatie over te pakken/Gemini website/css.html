CSS Referentie

De stylingtaal om je webpagina's vorm te geven, van basiskleuren tot complexe, responsive layouts.

1. CSS Basis: Syntax, Cascade & Specificiteit

Een CSS-regel bestaat uit een selector en een declaratieblok (property: value;). De cascade bepaalt welke stijlregel wint op basis van bron, specificiteit en volgorde. Inheritance laat sommige eigenschappen (zoals lettertype) overerven door kind-elementen.

Specificiteit (van hoog naar laag): Inline stijlen > ID > Class/Attribuut/Pseudo-class > Element/Pseudo-element. Gebruik !important met grote terughoudendheid.

2. Selectors

Patronen om elementen te selecteren voor styling.

  • Type, Class, ID: p, .mijn-klasse, #mijn-id
  • Attribuut: [type="email"], [href^="https"]
  • Pseudo-classes: :hover, :focus, :nth-child(n), :has()
  • Pseudo-elements: ::before, ::after, ::first-letter
  • Combinators: div p (descendant), div > p (child), h2 + p (adjacent sibling)

3. Waarden & Eenheden

De eenheden die je gebruikt om groottes en afstanden te definiÃ«ren.

  • Absoluut: px (pixels)
  • Relatief: %, em (relatief t.o.v. parent font-size), rem (relatief t.o.v. root font-size)
  • Viewport: vw (viewport width), vh (viewport height)
  • Functies: calc(), min(), max(), clamp()

4. Kleuren & Kleurruimtes

Manieren om kleuren te specificeren in CSS.

  • Hexadecimaal: #ff0000, #f00
  • RGB/RGBA: rgb(255, 0, 0), rgba(255, 0, 0, 0.5) (met transparantie)
  • HSL/HSLA: hsl(0, 100%, 50%) (Hue, Saturation, Lightness)
  • Moderne kleurruimtes: lab(), lch() voor meer perceptueel uniforme kleuren.

5. Box Model, Sizing & Overflow

Elk element is een doos, bestaande uit content, padding, border, en margin. Met box-sizing: border-box; zorg je ervoor dat de totale breedte en hoogte inclusief padding en border zijn, wat layouts vaak voorspelbaarder maakt.

6. Positionering & Z-index

De position-eigenschap bepaalt hoe een element in het document wordt geplaatst.

  • static: Normale flow (default).
  • relative: Verplaatsbaar t.o.v. zijn normale positie.
  • absolute: Verplaatsbaar t.o.v. het dichtstbijzijnde gepositioneerde voorouder-element.
  • fixed: Verplaatsbaar t.o.v. de viewport.
  • sticky: Een mix tussen relative en fixed, blijft "plakken" tijdens het scrollen.

z-index bepaalt de stapelvolgorde van gepositioneerde elementen.

7. Weergave & Layout

Moderne layout-technieken voor het bouwen van complexe en responsive interfaces.

Flexbox

Een eendimensionaal layoutsysteem, ideaal voor het uitlijnen van items in een rij of kolom. Gebruik display: flex; op een container. Belangrijke eigenschappen zijn justify-content (uitlijning op de hoofdas) en align-items (uitlijning op de kruisas).

Grid

Een tweedimensionaal layoutsysteem, perfect voor complexe layouts. Gebruik display: grid; op een container. Definieer kolommen en rijen met grid-template-columns en grid-template-rows.

8. Sizing & Containment

Eigenschappen om de grootte en rendering performance te beheersen.

  • aspect-ratio: Behoudt een vaste breedte-hoogteverhouding.
  • object-fit: Bepaalt hoe een <img> of <video> zijn container vult (cover, contain).
  • contain: Optimaliseert de rendering door de browser te vertellen dat een element onafhankelijk is van de rest van de pagina.

9. Achtergronden, Borders & Effecten

Visuele effecten om je ontwerp te verbeteren.

  • background-image: Gebruik een afbeelding of een linear-gradient().
  • border-radius: Ronde hoeken.
  • box-shadow: Een schaduw rondom een element.
  • filter: Visuele effecten zoals blur() of grayscale().
  • backdrop-filter: Past een filter toe op het gebied achter een element (voor "glassmorphism"-effecten).

10. Typografie & Tekst

Styling van tekst en lettertypen.

  • font-family: Specificeert het lettertype.
  • font-size: De grootte van de tekst (gebruik rem).
  • font-weight: De dikte van de letters (bv. 400 voor normaal, 700 voor vet).
  • line-height: De regelhoogte voor betere leesbaarheid.
  • color: De tekstkleur.

11. Transities & Animaties

Transities

Maakt een geleidelijke overgang tussen twee staten (bv. bij :hover). Gebruik de transition-eigenschap.

Animaties

Voor complexere, op keyframes gebaseerde animaties. Definieer de animatie met @keyframes en pas deze toe met de animation-eigenschap.

  --primary-color: #3b82f6;

  background-color: var(--primary-color);

13. Media Queries

De kern van responsive design. Hiermee pas je stijlen toe op basis van de eigenschappen van het apparaat, zoals de breedte van het scherm of gebruikersvoorkeuren.

/* Stijlen voor schermen breder dan 768px */
@media (min-width: 768px) {
  .container { max-width: 720px; }
}

/* Stijlen voor donkere modus */
@media (prefers-color-scheme: dark) {
  body { background: #111; color: #eee; }
}

14. Container Queries

Een moderne techniek waarmee je stijlen kunt aanpassen op basis van de grootte van een container-element, in plaats van de hele viewport. Activeer dit op een element met container-type.

.card-container {
  container-type: inline-size;
}

@container (min-width: 500px) {
  /* Stijlen voor als de container breder is dan 500px */
}

15. Scrolling & Interactie

Eigenschappen om de scroll-ervaring en gebruikersinteractie te verbeteren.

  • scroll-behavior: smooth;: Zorgt voor een vloeiende scroll-animatie bij ankerlinks.
  • scroll-snap-type: CreÃ«ert "snap points" voor carrousels en scroll-interfaces.
  • pointer-events: none;: Zorgt ervoor dat een element geen muis-events meer ontvangt.

16. Toegankelijkheid & UI

CSS-eigenschappen die direct bijdragen aan een betere gebruikerservaring en toegankelijkheid.

  • :focus-visible: Stijlt de focus-ring alleen voor toetsenbordgebruikers, niet bij muisklikken.
  • accent-color: Stelt de kleur in van UI-elementen zoals checkboxes en radio buttons.
  • prefers-reduced-motion: Een media query om animaties te verminderen voor gebruikers die dit prefereren.

17. Print & Paged Media

18. @-regels

Speciale regels in CSS die instructies geven aan de browser.

  • @import: Importeert een ander stylesheet (minder performant dan <link>).
  • @media: Definieert een media query.
  • @keyframes: Definieert een animatie.
  • @font-face: Laadt een custom lettertype.
  • @layer: Definieert een cascade layer voor betere controle over specificiteit.

19. Performance & Best Practices

Tips voor het schrijven van efficiÃ«nte en onderhoudbare CSS.

  • Houd de specificiteit zo laag mogelijk.
  • Gebruik cascade layers om je stylesheet te structureren.
  • Vermijd dure eigenschappen zoals filter en box-shadow in animaties waar mogelijk.
  • Gebruik will-change spaarzaam om de browser te hinten over komende animaties.
